Guide d'installation Windows

Installateur (Certificat) : Si vous avez l'erreur "Besoin d'un certificat", suivez ces étapes :
  • Clic droit sur le fichier .msix -> Propriétés.
  • Onglet Signatures numériques -> Sélectionner la ligne -> Détails.
  • Afficher le certificat -> Installer le certificat.
  • Choisir Machine locale -> Placer dans : Personnes approuvées.
  • Validez et relancez l'installation.

Indicate de confiance : L'application n'utilise aucune donnée personnelle externe.
